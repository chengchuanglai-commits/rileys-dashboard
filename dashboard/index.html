⚙️ 如何配置 Google Analytics 自动同步 → 点击展开教程
  1. 进入 Google Analytics → 左下角「管理（齿轮）」→「媒体资源设置」，找到 媒体资源 ID（纯数字，例如 123456789）
  2. 打开 Google Apps Script → 点击「新建项目」→ 删除默认代码，把下方 全部代码 粘贴进去，把第一行的 YOUR_PROPERTY_ID 替换成你的 ID

  5. 复制显示的「网络应用 URL」（以 https://script.google.com/macros/s/ 开头）
  6. 回到这里，点击上方 「设置 API」，粘贴 URL 保存 ✅
