## 关键发现

1. **修成复利后,B/D/E/F/G 的'领先'缩水**:旧口径 B+$172/F+$160 排最前;复利后 B+5.0%/F+4.7% 掉到中游。**验证了它们之前的领先有口径水分。**

2. **C 仍第一(+7.1%)且去尾2笔后仍正**,但 n=15小、80%胜率可能运气。

3. **B 去尾2笔后最扛($2067)**,收益不靠少数几笔——意外地比D/E/F稳(但仍要退役,研究价值已榨干)。

4. **⚠️SPY同期-0.6%(平盘)**:所有'+4~7%'领先仍是平盘噪音,统一口径让排名公平了但没改变'平盘测不出真edge'。

5. **真正要押的可交易腿(MOM-MA/MOM-H)0平仓没出数**——有数据的全是要退役/不可交易的,别被这张表的'谁第一'带跑。
