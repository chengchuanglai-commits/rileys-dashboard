# 金融晨报模块设计文档

**日期：** 2026-05-18
**状态：** 已确认，待实施

---

## 概述

在 Riley's Dashboard 中新增金融晨报模块，每天自动生成市场概况、宏观指标解读，以及全市场 AI 筛选的 5 支股票买卖时机判断，并将每日数据存档以供历史查询。

---

## 一、位置与布局

### Dashboard 整体结构变化

```
Row 1: 账户总览 · 今日任务 · 运动计划（不变）
Row 2: 项目追踪（不变）
Row 3: 📊 金融晨报（新增，独立全宽行）
Row 4: 新闻 · 股票组合（原有，下移）
```

### 晨报卡片内部布局

两栏式结构：

```
┌─────────────────────────────────────────────────────────────────┐
│ 📊 金融晨报              🗓 2026-05-18  ● 盘前  最后更新 07:30   │
├───────────────┬─────────────────────────────────────────────────┤
│  关键数据      │  [ 今日晨报 ] [ 📈 买卖时机 ]                    │
│               │                                                 │
│  S&P 500 期货 │  Tab 1：今日晨报正文（市场概况/宏观/财报/建议）    │
│  ↓ 0.4%       │  Tab 2：5支推荐股票卡片（买入区/目标/止损/理由）   │
│               │                                                 │
│  10Y 美债     │                                                 │
│  4.55%        │                                                 │
│               │                                                 │
│  EPS 超预期率 │                                                 │
│  84%          │                                                 │
└───────────────┴─────────────────────────────────────────────────┘
```

**左栏（200px）：** 3 个关键数字卡片，带颜色（红/绿/黑）和副标题说明
**右栏（flex 1）：** Tab 切换，今日晨报 + 买卖时机

---

## 二、内容规格

### 关键数字（左栏，固定 3 项）

| 指标 | 说明 | 颜色规则 |
|------|------|----------|
| S&P 500 期货涨跌幅 | 百分比，带方向箭头 | 正数绿 / 负数红 |
| 10年期美债收益率 | 百分比 + 变化 bps | 固定黑色 |
| Q1 EPS 超预期率 | 百分比 | 固定绿色 |

### Tab 1：今日晨报

四个版块，按固定顺序：
1. **市场概况** — 主要指数期货走势，今日关注焦点
2. **宏观焦点** — 利率、通胀、美联储、地缘事件
3. **财报亮点** — 本周重要财报、Q1季报季整体数据
4. **交易建议** — 2-3 条简短的方向性建议（非具体股票）

每版块：标签 + 正文（120字以内），适合投资新手阅读。

### Tab 2：买卖时机

**每日固定 5 支股票**，由 Agent 从全市场各板块筛选。

每支股票展示：

| 字段 | 说明 |
|------|------|
| 股票代码 + 名称 | 如 NVDA · Nvidia |
| 板块 | 如 科技/半导体 |
| 建议方向 | 买入 ✓ / 卖出 ✗ / 观望 ⏳（带颜色标签）|
| 买入区间 | 价格区间，如 $900–$920 |
| 目标价 | 上行目标 |
| 止损价 | 下行止损 |
| 理由 | 一句话说明，新手友好 |

**板块覆盖原则：** 5 支股票尽量覆盖不同板块（科技/能源/医疗/消费/固收），避免集中单一板块。

## 三、数据架构

### 文件结构

```
dashboard/
├── index.html                      ← 现有文件，新增晨报卡片
├── morning-note.json               ← 今日数据（每天 07:30 覆盖）
└── morning-note-history/
    ├── 2026-05-18.json             ← 历史存档（同时写入，按日期命名）
    ├── 2026-05-17.json
    └── ...
```

### morning-note.json 结构

```json
{
  "date": "2026-05-18",
  "generated_at": "2026-05-18T07:30:00",
  "market_status": "pre-market",
  "market": {
    "sp500_futures_pct": -0.4,
    "treasury_10y": 4.55,
    "treasury_10y_change_bps": 9,
    "eps_beat_rate": 84
  },
  "note": {
    "market_overview": "S&P 500 期货下跌 0.4%，延续上周五跌势...",
    "macro": "10年期美债升至 4.55%，加息概率升至 45%...",
    "earnings": "84% 公司 EPS 超预期，幅度 +18.2%...",
    "trade_ideas": "谨慎科技多单；关注能源板块；NVDA 期权机会..."
  },
  "stock_picks": [
    {
      "ticker": "NVDA",
      "name": "Nvidia",
      "sector": "科技/半导体",
      "direction": "watch",
      "buy_zone": "$900–$920",
      "target": "$980",
      "stop_loss": "$870",
      "reason": "财报前 IV 高企，等财报方向确认后再入场"
    }
  ]
}
```

`direction` 枚举值：`"buy"` / `"sell"` / `"watch"`

### 数据更新方式

使用 `schedule` 技能创建定时 Agent，每天早上 **07:30（北京时间）** 自动执行：

1. 搜索当日市场数据（期货、美债、财报数据）
2. 搜索全市场热点，筛选 5 支股票
3. 调用 Claude API 生成晨报文本 + 买卖判断
4. 写入 `dashboard/morning-note.json`
5. 同时存档到 `dashboard/morning-note-history/YYYY-MM-DD.json`

## 四、Dashboard 代码修改

### index.html 修改范围

1. **新增 CSS**：晨报卡片样式（Tab 栏、股票卡片、方向标签颜色）
2. **新增 HTML**：Row 3 晨报卡片结构，插入现有 Row 2（项目追踪）之后
3. **新增 JS**：
   - `loadMorningNote()` — 页面加载时 fetch `morning-note.json` 并渲染
   - `renderStockPicks(picks)` — 渲染 5 支股票卡片
   - `switchTab(tab)` — 切换今日晨报 / 买卖时机
   - 数据缺失时显示占位状态（"今日晨报生成中..."）

### 不修改的内容

现有所有模块（账户、任务、运动、项目、新闻、股票组合）代码不变，新增的晨报卡片作为独立模块插入。

## 五、用户体验细节

- **新手友好**：所有文字带上下文解释，数字旁有副标题说明含义
- **颜色语义**：红色=下跌/风险，绿色=上涨/超预期，黑色=中性数据
- **数据缺失处理**：`morning-note.json` 不存在时显示"今日晨报生成中，请稍后刷新"
- **手动刷新**：卡片右上角 ↻ 按钮，点击重新 fetch JSON（不触发 Agent）
- **历史查询**：预留扩展口，未来可在 UI 上加日期选择器读取历史存档
