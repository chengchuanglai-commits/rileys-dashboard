# 杠杆指数腿 设计文档 (Leverage Index Leg)

**日期**: 2026-07-07
**目标**: 用近凯利最优、纪律化的 1.5x 杠杆持有指数(QQQ),靠两道降风险信号在危险时自动退杠,先在模拟盘前向验证执行机制,再上真钱——作为"做大资金池"的增长引擎。

---

## 1. 背景与理由(为什么这么设计)

Riley 想加速做大 $2000 资金池,提出用杠杆。核心决策依据:

- **杠杆不创造 edge,只放大已有 edge**。主动腿 edge 未证明→不给主动腿上杠杆(=纯赌)。**只对指数上杠杆**,因为股权溢价是真实、可测的 edge。
- **"风险越大收益越大"在杠杆上是错的**。几何增长 g ≈ L·μ − (L²σ²)/2,波动率拖累随杠杆平方增长。存在凯利最优 L\*=μ/σ²,对指数约 1.5–2x;**超过就是更多风险、更少增长**。回测证据:3x(+34% CAGR/83%回撤)几乎没比 2x(+32%/65%)多赚。
- **Riley 风险画像**: 可全亏 $2000(接受归零);~40% 回撤可承受;想在明确信号下最大化收益。
- **回测(QQQ 2020-01→2026-07,含2020崩盘+2022熊市,$2000本金)选定 1.5x+200MA闸门**: +28% CAGR / 42% 最大回撤($2000→~$10k),回撤贴 Riley 的 40% 线,略低于凯利最优换更平滑的持有体验(拿得住比理论最优更重要)。

## 2. 策略规则

| 部分 | 规则 |
|---|---|
| 标的 | QQQ |
| 基础杠杆 | **1.5x** |
| 信号1·趋势闸门 | QQQ 收盘 **> 200日SMA → 1.5x**;跌破 → **1x**(用前一日信号,防前视) |
| 信号1b·whipsaw确认 | 跌破后要 **连续站上200MA 5天(confirm_days=5)才重新加杠**(降杠即时、加杠要确认)。2026-07-07回测:改善2008/2020震荡崩盘的"假反弹被割",全程收益几乎无损 |
| 信号2·波动闸门 | 20日已实现年化波动 **> 30% → 强制退回 1x**(危险时降杠,不赌方向) |
| 融资成本 | 借入的 (L−1) 部分按 **6%/年**(日 = 0.06/252)扣,真实口径 |
| 再平衡 | 实际杠杆漂出 **[1.3x, 1.7x]** 带宽 或 信号档位切换 时才调整(减少换手/波动损耗) |
| 基准 | **1x QQQ 买入持有**(衡量杠杆的净增益与回撤代价) |

> 波动阈值 30%、再平衡带宽 [1.3,1.7] 为起始值,在实施阶段的深化回测中调优(见 §4)。

## 3. 架构(paper 腿)

- 新增模拟腿 `lev`,产出 `data/portfolio_lev.json` + `dashboard/portfolio-lev.js`,像其它腿一样每日回填。
- **回填脚本** `backfill-portfolio-lev.py`: 读 QQQ 日线 → 算 200MA/20日波动 → 逐日定杠杆档 → 应用杠杆日收益(扣融资成本)+再平衡带宽 → 输出净值曲线 + 指标。
- **关键指标**: 净值、CAGR、**最大回撤**、当前杠杆档位、vs 1x QQQ 的净增益。
- Dashboard 加一张 `lev` 卡片显示上述指标。
- 数据流: QQQ日线(yfinance/FMP) → backfill-portfolio-lev.py → portfolio_lev.json → dashboard。与现有各腿回填模式一致。

## 4. 验证 + 上真钱路径

1. **深化回测**(实施第一步): 把波动闸门叠进回测;拉多个历史窗口——**含 2000-2002 纳指-78%、2008-09 等灾难期**——确认最坏情况不归零、参数非过拟合、42%回撤是否在别的窗口失守。
2. **paper 前向跑几周**: 验证"200MA切换/波动降杠/再平衡"执行干净、不出 bug、档位切换及时。
3. **执行稳 + 真钱闸门齐 → 上真钱**: 比主动腿更快能上(指数 edge 是真的,不需攒80笔证明)。真钱用 margin 账户,注意保证金机制。

## 5. 诚实红线(风控与已知局限)

- **回测 42% 回撤是乐观值**: 2020-26 无真崩盘。真遇 -50% 崩盘,1.5x ≈ -75%,虽不归零但极痛;-67% 崩盘 → 1.5x 归零。这是为什么要跑灾难窗口(§4.1)。
- **强平风险**: 1.5x 相对抗强平(QQQ需跌~33%+才碰典型维持保证金),且 200MA 闸门会提前降杠;但真钱 margin 账户仍需监控维持保证金,避免在底部被强平。
- **波动损耗**: 每日/频繁再平衡的杠杆在震荡市磨损;再平衡带宽设计即为缓解。
- **不梭哈**: 这是"输得起的钱 + near-Kelly + 纪律化"的杠杆,不是最大化杠杆。杠杆档位永不超 1.5x。
- **只对指数**: 绝不对未证明 edge 的主动腿上杠杆。

## 6. 成功标准

- 深化回测在含灾难期的多窗口下:不归零、回撤可接受、相对 1x 有净增益、参数稳健(非过拟合)。
- paper 前向:执行机制干净无 bug、档位切换正确、净值曲线合理。
- 达标后:配合真钱闸门,决定是否/如何上真钱。
