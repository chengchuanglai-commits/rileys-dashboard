# 读财报训练路径（给 20% 长线持股用）

> 目的：为"20% 长线看好的股"做基本面判断。读财报对短线波动没用，只对**长期持有一家公司**有意义。
> 原则：散户读财报选股长期也难赢指数（统计困境），但它有动量没有的优势——**你买是因为理解，不是因为回测，没有幸存者偏差的自欺**。而且看懂商业/风险的技能本身值得学。

## 三张表，先搞懂"它们各回答什么"

| 报表 | 回答的问题 | 长线投资最关键的一点 |
|---|---|---|
| **利润表 (Income Statement)** | 公司赚不赚钱？ | 收入增长 + 利润率趋势（不是单期数字，是**趋势**） |
| **资产负债表 (Balance Sheet)** | 公司稳不稳、会不会暴雷？ | 负债/现金（**会不会借太多钱死掉**） |
| **现金流量表 (Cash Flow)** | 赚的是不是真钱？ | 经营现金流 vs 净利润（**利润是不是纸面的**） |

> 黄金法则：**利润可以造假，现金流很难**。先看现金流量表，再看利润表。

## 阶段1：先学会问 6 个问题（不用懂会计，先建直觉）

拿任何一家你想长期持有的公司，回答：

1. **收入这几年在涨吗？** → 利润表，看 3-5 年 Revenue 趋势。**降的直接淘汰**。
2. **它真赚钱吗？还是烧钱？** → 看净利润 + **经营现金流（Operating Cash Flow）是不是正的**。
3. **利润是真钱吗？** → 经营现金流 ≈ 或 > 净利润 = 健康；远小于 = 利润有水分（应收账款堆积/做账）。
4. **它欠了多少钱？会被债务压垮吗？** → 资产负债表，看 总负债 vs 现金 + 经营现金流够不够还。
5. **赚的钱花哪了？** → 回购/分红（对股东好）vs 不停增发稀释（对股东坏）。
6. **护城河在哪？** → 毛利率高且稳 = 有定价权/护城河（如 50%+ 毛利率常是好生意）。

**6 个问题里有 2-3 个明显不行 → 不碰。** 这一步就能筛掉大部分坑。

## 阶段2：关键指标（学会算/查，5个够用）

| 指标 | 怎么看 | 警戒 |
|---|---|---|
| **营收增速 (Revenue Growth)** | 同比、且连续多季 | 持续放缓/转负 = 警惕 |
| **毛利率 (Gross Margin)** | 越高越有护城河，看趋势 | 持续下滑 = 竞争加剧/失去定价权 |
| **经营现金流 (OCF)** | 必须为正且 ≈净利润 | OCF 长期 < 净利润 = 利润有水分 |
| **净负债 (Net Debt)** | 总债务 − 现金 | 高 + OCF 弱 = 暴雷风险（SVB/BBBY 都是）|
| **ROE / ROIC** | 资本回报率，>15% 是好生意 | 持续低 = 赚钱效率差 |

## 阶段3：上手练（边做边学）

1. **从你懂的公司开始**——你用过它产品、理解它生意的（不是听说赚钱）。
2. **每家公司花 30 分钟**：6 个问题过一遍 + 5 个指标查一遍（雪球/财报网站直接有）。
3. **写一句话投资逻辑**：「我买它，因为 ___；我会卖，如果 ___」。写不出来 = 还不理解，别买。
4. **长期跟踪**：每季度财报出来，看那 5 个指标的**趋势**有没有变坏。

## 红线（看到就远离）

- 经营现金流持续为负 + 不停增发融资（烧钱续命，BBBY 模式）
- 收入下滑 + 负债攀升（衰退中的高杠杆，SVB 模式）
- 利润涨但经营现金流不涨（利润注水）
- 管理层频繁变更 + 财报频繁重述（治理问题）

## 数据/工具

- **雪球/同花顺**：中文，财报数据 + 关键指标现成（适合起步）
- **FMP API**：我们已有（key 在 secret），能拉 ratios/key-metrics/income-statement，可做成"输入代码→自动给6问5指标体检表"的工具
- **公司财报原文**：SEC EDGAR（美股 10-K/10-Q），进阶用
